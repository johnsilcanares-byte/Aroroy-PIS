# Aroroy-PIS (Patient Information System)
Overview

Aroroy-PIS (Patient Information System) is a web-based healthcare management system developed to streamline the management of patient information, appointments, medicine inventory, and report generation for healthcare facilities. The system enables healthcare personnel to efficiently organize medical records, monitor medicine stocks, schedule patient appointments, and generate essential reports through an intuitive and user-friendly interface.
